Human Computer Interaction - Webdesign

The goal of this project was to design an action website for a 'new' brand item. We had the choice from three different brands.

I chose for the brand The North Face. This action website revolves around a new backpack. You can adjust the colors, embroidery and you can choose different technological gadgets that can be added to the backpack.

This project is in Dutch.
